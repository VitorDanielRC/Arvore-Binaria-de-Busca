Estruturas de Dados – Árvore Binária de Busca

Implementação e Visualização com Apoio de Python e Grafos

Aplicação Web Interativa
1. Lista inicial de valores

Informe aproximadamente 25 números inteiros (ordenados ou não). Os dados serão enviados para o backend em Python, que construirá uma árvore binária de busca balanceada.

Caso a entrada fique em branco, será utilizada uma lista padrão definida no programa em Python.

2. Operações sobre a árvore

Após a construção, é possível solicitar ao backend as operações de inserção, busca + remoção e remoção direta. A cada alteração os dados retornam em formato JSON e a visualização é atualizada no navegador.

Descrição do Trabalho
Objetivo

Desenvolver uma aplicação web que integre Python e HTML/JavaScript, capaz de construir e manipular uma árvore binária de busca a partir de dados fornecidos pelo usuário, utilizando conceitos de grafos para a visualização.

Integração Python & Front-end
  • O backend, implementado em Python (Flask), mantém a lógica da BST.
  • As operações são expostas por meio de uma API REST simples em JSON.
  • O front-end envia e recebe dados usando fetch() e redesenha a árvore em um elemento <canvas>.
Operações Disponíveis
  • Inicialização: /api/init – recebe a lista de números e devolve a árvore balanceada.
  • Inserção: /api/inserir – inclui novo elemento e reequilibra a estrutura.
  • Busca + remoção: /api/procurar_remover – remove o valor, caso encontrado.
  • Remoção direta: /api/remover – exclui o valor informado, se existir.
Resultados Esperados

A aplicação ilustra, de maneira prática, os impactos do balanceamento sobre a altura da árvore e o custo das operações de busca, ao mesmo tempo em que reforça a integração entre estruturas de dados e desenvolvimento web.
